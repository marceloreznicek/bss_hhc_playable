# PRÉ-CODE — HeroMerge playable (merge × auto-battler)

## TAREFA

Estamos prestes a implementar **HeroMerge**, um playable de anúncio (HTML5
single-file, formato 1080×1920 portrait) construído sobre o motor existente
do **HeroSwarm v2**. O HeroSwarm v2 já foi shippado e funciona — vamos clonar
sua master HTML e modificar.

Vocês são 8 IAs especialistas. **Antes de qualquer linha de código**, revisem
o plano abaixo e apontem:
- Edge cases que vão quebrar o jogo
- Race conditions na interação drag-merge
- Problemas de UX na mecânica magnética
- Riscos de regressão no motor reusado
- Bugs prováveis no animation timing
- Soluções alternativas que considerem melhores

Não inventem features fora do escopo do GDD v0.1. **Mudança mínima, máxima
clareza.** O playable inteiro precisa caber em ~310 LoC novos.

## GDD CONSOLIDADO (HeroMerge v0.1)

### Conceito (1 parágrafo)

Tap **SUMMON** dropa Recruits Tier-1 no campo. **Drag** uma unit em cima de
outra do mesmo tier → mescla em tier maior. Ao tocar uma unit, todas do
mesmo tier num raio de ~350px são puxadas magneticamente em direção ao dedo.
Ao soltar (ou após 0.6s parado), o cluster funde em UMA unit do próximo
tier no ponto da soltada. Combate auto. Sobreviver as 3 waves do HeroSwarm
e matar o Demon Lord = vitória.

### Ladder de tiers (5 níveis, reusa sprites do HeroSwarm)

| Tier | Nome | Sprite | HP | DMG | Speed |
|---|---|---|---|---|---|
| 1 | Recruit    | hero    (intro-hero.png)  | 5   | 1  | 110 |
| 2 | Lumberjack | ogre    (lenhador.png)    | 10  | 3  | 70  |
| 3 | Knight     | knight  (igris.png)       | 20  | 6  | 100 |
| 4 | Brute      | brute   (juggermon2.png)  | 35  | 4  | 60  |
| 5 | Champion   | dragon  (overlord.png)    | 120 | 6  | 80  |

(Stats vêm direto das tabelas existentes — apenas reuso.)

### Mecânica magnética — comportamento exato

- Player toca uma unit → guarda `STATE.heldActor`
- Cada frame: outras units do MESMO tier dentro do raio R=350px lerpam em
  direção ao `heldActor` na velocidade ~600 px/s (ease-out)
- `heldActor` segue o dedo (pointer x/y → game coords)
- Ao soltar (pointerup) OU após 600ms de hold-still:
  - Identifica o cluster: heldActor + todas units do mesmo tier dentro de
    R_MERGE=80px da posição do dedo
  - Se cluster.length >= 2: despawn todas + spawn 1 unit do próximo tier no
    ponto do dedo + flash + SFX
  - Se cluster.length < 2: heldActor volta pro lugar (cancelar)

### SUMMON button (substitui as 3 cards)

- 1 botão grande na parte de baixo da tela
- Cooldown ~700ms (estado visual: cd-sweep + cd-text)
- Spawna 1 Recruit (Tier 1) em posição aleatória dentro do bottom half
  do battlefield (mesmo `bottomHalfY` usado pra enemies)
- Pode ser tapado a qualquer momento, mesmo durante drag-merge

### Tutorial (3 passos)

1. **0–3s**: bubble "TAP SUMMON!" + guide hand pulsa no botão
2. **Após 2 Recruits estarem vivos**: bubble "DRAG TO MERGE!" + guide hand
   demonstra um drag de Recruit pra Recruit (a animação magnética faz o
   ensino acontecer visualmente)
3. **Após 1ª merge feita**: tutorial off

### Win / fail / endcard

Inalterados do HeroSwarm v2 (Demon Lord killed = win, Fire Breath = fail,
endcard = mesmo intro-hero portrait + "PLAY NOW TO BUILD YOUR ARMY").

### 4. Drag-to-merge — pointer events (~80 LoC)

```js
var STATE = { ...
  heldActor: null,
  pointerX: 0, pointerY: 0,
  holdStartedAt: 0,
};

document.getElementById('lane').addEventListener('pointerdown', function(e) {
  var hit = findActorAt(toGameCoords(e));
  if (!hit || hit.tier == null || hit.tier >= 5) return;
  STATE.heldActor = hit;
  STATE.pointerX = toGameCoords(e).x;
  STATE.pointerY = toGameCoords(e).y;
  STATE.holdStartedAt = performance.now();
  hit.state = 'held';
});

document.addEventListener('pointermove', function(e) {
  if (!STATE.heldActor) return;
  var p = toGameCoords(e);
  STATE.pointerX = p.x; STATE.pointerY = p.y;
  STATE.heldActor.x = p.x; STATE.heldActor.y = p.y;
  STATE.holdStartedAt = performance.now();  // reset auto-fire
});

document.addEventListener('pointerup', function() {
  if (!STATE.heldActor) return;
  resolveMergeAtPointer();
  STATE.heldActor = null;
});
```

### 5. Atração magnética (~50 LoC, no game loop)

```js
function tickMagnetic(dt) {
  var h = STATE.heldActor;
  if (!h) return;
  var R = 350, R2 = R * R;
  for (var i = 0; i < STATE.units.length; i++) {
    var u = STATE.units[i];
    if (u === h || u.tier !== h.tier || u.dead) continue;
    var dx = STATE.pointerX - u.x, dy = STATE.pointerY - u.y;
    var d2 = dx*dx + dy*dy;
    if (d2 > R2) continue;
    var d = Math.sqrt(d2);
    if (d < 1) continue;
    var pull = Math.min(600 * dt, d);  // px/s, never overshoot
    u.x += (dx / d) * pull;
    u.y += (dy / d) * pull;
    applyActorTransform(u);
  }
  // Auto-fire merge if held still for >600ms
  if (performance.now() - STATE.holdStartedAt > 600) {
    resolveMergeAtPointer();
    STATE.heldActor = null;
  }
}
```

### 6. Merge resolution (~40 LoC)

```js
function resolveMergeAtPointer() {
  var h = STATE.heldActor;
  if (!h) return;
  var R_MERGE = 80, R2 = R_MERGE * R_MERGE;
  var cluster = [h];
  for (var i = 0; i < STATE.units.length; i++) {
    var u = STATE.units[i];
    if (u === h || u.tier !== h.tier || u.dead) continue;
    var dx = STATE.pointerX - u.x, dy = STATE.pointerY - u.y;
    if (dx*dx + dy*dy < R2) cluster.push(u);
  }
  if (cluster.length < 2) {
    h.state = 'idle';
    return;  // cancelado, fica onde está
  }
  // Despawn cluster
  cluster.forEach(function(u) { u.dead = true; u.dom.remove(); });
  STATE.units = STATE.units.filter(function(u) { return !u.dead; });
  // Spawn next tier no ponto do dedo
  var nextKey = TIER_TYPE[h.tier];  // tier 1 → index 1 → ogre
  if (!nextKey) return;             // já tava no tier 5
  var newU = createActor(nextKey, UNIT_TYPES, STATE.pointerX, STATE.pointerY);
  newU.tier = h.tier + 1;
  STATE.units.push(newU);
  playSFX('unlock');  // power up sound
  // TODO: flash visual no spawn point (CSS .merge-flash)
}
```
