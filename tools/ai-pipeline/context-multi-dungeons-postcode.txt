## PLANO ORIGINAL (Kimi K2.6 — síntese aprovada pelo usuário)

# Kimi K2.6

- Modelo: `moonshotai/kimi-k2.6`

---

 1. **Causa raiz consolidada**
A feature fragmenta a fonte de verdade entre CSV, SOs gerados e estado legado não serializável por `JsonUtility`, enquanto expõe o cálculo do `tideIdx` e o desbloqueio de dungeons a condições de corrida entre preview, execução e persistência.

2. **Plano priorizado**
1. **Save v2 atômico**: Substituir `deepestFloor` global por `List<DungeonProgressEntry>` serializável, rebuild de `Dictionary` em `Load()`, e encapsular atualização de deepest + unlocks + `seenJustUnlocked` em `FinalizeRunTransaction()` com único `Save()` antes de disparar callbacks de UI.
2. **Tide determinístico**: Criar `GetTideIndex(dungeonId, floor, enemyCount)` pura com hash FNV-1a (nunca `string.GetHashCode()`); garantir que `PreviewNextFloor` e `BeginFloor` invoquem a mesma função com argumentos idênticos.
3. **Registry runtime simples**: Parser direto de CSV para `Dictionary<string, DungeonDef>` no bootstrap, sem `[InitializeOnLoad]` nem geração de SO; validar ranges e IDs únicos no parse.
4. **CombatResolver com regra injetada**: Passar `DungeonRule` imutável; aplicar mis-focus por tick no ataque inimigo sem mutar `EnemyStats`; resetar `FocusEnemyIndex = -1` a cada `StartRun`/floor; bloquear foco em `NoFocus` tanto na UI quanto no resolver.
5. **UI por último**: Implementar Dungeon Select e badge "New" somente após validar persistência e determinismo; badge limpa ao entrar no roster do dungeon.

3. **Conflitos entre IAs**
- **Gerar SO em edit-time vs parser runtime**: Opus/Gemini sugerem SOs + reimportação manual; Qwen/KAT/GPT55 defendem parser runtime puro. O lado runtime tem mais evidência técnica para protótipo: elimina race de domain reload, merge conflicts e duplicação de verdade.
- **HMAC/checksum no save**: Sonnet/Grok exigem hash hardcoded; outras tratam como dívida técnica futura. Como não há backend nem monetização ainda, a evidência favorece protótipo sem checksum, mas deve ser registrado como débito crítico antes de rankings ou recompensas.
- **Interface `IDungeonTuning` vs struct**: KAT propõe interface para desacoplar; maioria (Opus, GPT55, Qwen) prefere struct/enum puro. Struct é mais adequada por ser serializável, imutável e sem overhead de alocação no loop de combate.

4. **Convergências**
- **Serialização JsonUtility**: todas as 8 IAs concordam em abandonar `Dictionary`/HashSet direto e usar `List<DungeonProgressEntry>` com rebuild em `Load()`, sanitizando duplicatas.
- **Transação atômica**: consenso absoluto de que unlock/deepest/gold devem ser computados e salvos em bloco único antes de qualquer callback de UI, evitando perda de estado em crash ou force-close.
- **Hash estável do Tide**: unanimidade em rejeitar `string.GetHashCode()` e centralizar cálculo em função pura compartilhada entre preview e run real.
- **Isolamento de estado**: concordam que `FocusEnemyIndex` deve resetar a cada run/floor e que `CombatResolver` não pode depender de estado mutável de `RunController` para decidir regras.

using System.Collections.Generic;
using System.IO;
using System.Text;
using UnityEngine;

namespace DiverDungeon.Prototype
{
    public static class CsvLoader
    {
        public static List<Dictionary<string, string>> LoadFile(string fileName)
        {
            string path = ResolveCsvPath(fileName);
            if (path == null)
            {
                Debug.LogWarning($"[CsvLoader] CSV not found: {fileName}");
                return new List<Dictionary<string, string>>();
            }
            string text;
            try { text = File.ReadAllText(path, Encoding.UTF8); }
            catch (System.Exception e)
            {
                Debug.LogWarning($"[CsvLoader] Read failed for {path}: {e.Message}");
                return new List<Dictionary<string, string>>();
            }
            return Parse(text);
        }

        public static string ResolveCsvPath(string fileName)
        {
            string streaming = Path.Combine(Application.streamingAssetsPath, "Balance", fileName);
            if (File.Exists(streaming)) return streaming;

            string projectRoot = Path.GetFullPath(Path.Combine(Application.dataPath, ".."));
            string atRoot = Path.Combine(projectRoot, "Balance", fileName);
            if (File.Exists(atRoot)) return atRoot;

            return null;
        }

        // Minimal RFC 4180-ish parser: supports quoted fields with embedded commas and ""-escaped quotes.
        public static List<Dictionary<string, string>> Parse(string text)
        {
            var result = new List<Dictionary<string, string>>();
            if (string.IsNullOrEmpty(text)) return result;

            var rows = new List<List<string>>();
            var currentRow = new List<string>();
            var currentField = new StringBuilder();
            bool inQuotes = false;

            for (int i = 0; i < text.Length; i++)
            {
                char c = text[i];
                if (inQuotes)
                {
                    if (c == '"')
                    {
                        if (i + 1 < text.Length && text[i + 1] == '"') { currentField.Append('"'); i++; }
                        else inQuotes = false;
                    }
                    else currentField.Append(c);
                }
                else
                {
                    if (c == '"') inQuotes = true;
                    else if (c == ',') { currentRow.Add(currentField.ToString()); currentField.Length = 0; }
                    else if (c == '\n')
                    {
                        currentRow.Add(currentField.ToString()); currentField.Length = 0;
                        rows.Add(currentRow); currentRow = new List<string>();
                    }
                    else if (c == '\r') { /* skip */ }
                    else currentField.Append(c);
                }
            }
            if (currentField.Length > 0 || currentRow.Count > 0)
            {
                currentRow.Add(currentField.ToString());
                rows.Add(currentRow);
            }

            if (rows.Count < 2) return result;
            var headers = rows[0];
            for (int r = 1; r < rows.Count; r++)
            {
                var row = rows[r];
                if (row.Count == 1 && string.IsNullOrEmpty(row[0])) continue;
                var dict = new Dictionary<string, string>(headers.Count);
                for (int c = 0; c < headers.Count; c++)
                {
                    string key = headers[c].Trim();
                    string val = c < row.Count ? row[c] : string.Empty;
                    dict[key] = val;
                }
                result.Add(dict);
            }
            return result;
        }

        public static int GetInt(Dictionary<string, string> row, string key, int fallback = 0)
        {
            if (row != null && row.TryGetValue(key, out var s) && int.TryParse(s, System.Globalization.NumberStyles.Integer, System.Globalization.CultureInfo.InvariantCulture, out var v))
                return v;
            return fallback;
        }

        public static float GetFloat(Dictionary<string, string> row, string key, float fallback = 0f)
        {
            if (row != null && row.TryGetValue(key, out var s) && float.TryParse(s, System.Globalization.NumberStyles.Float, System.Globalization.CultureInfo.InvariantCulture, out var v))
                return v;
            return fallback;
        }

        public static string GetString(Dictionary<string, string> row, string key, string fallback = "")
        {
            return row != null && row.TryGetValue(key, out var s) ? s : fallback;
        }

        public static Color ParseHexColor(string hex, Color fallback)
        {
            if (string.IsNullOrEmpty(hex)) return fallback;
            if (ColorUtility.TryParseHtmlString(hex, out var c)) return c;
            return fallback;
        }
    }
}

using System.Collections.Generic;
using UnityEngine;

namespace DiverDungeon.Prototype
{
    public static class DungeonRegistry
    {
        const string CsvFile = "balance_dungeons.csv";

        static List<DungeonDefinition> _dungeons;
        static Dictionary<string, DungeonDefinition> _byId;

        public static IReadOnlyList<DungeonDefinition> All
        {
            get { EnsureLoaded(); return _dungeons; }
        }

        public static DungeonDefinition GetById(string id)
        {
            EnsureLoaded();
            return id != null && _byId.TryGetValue(id, out var d) ? d : null;
        }

        public static void Reload()
        {
            _dungeons = null;
            _byId = null;
            EnsureLoaded();
        }

        static void EnsureLoaded()
        {
            if (_dungeons != null) return;
            _dungeons = new List<DungeonDefinition>();
            _byId = new Dictionary<string, DungeonDefinition>();

            var rows = CsvLoader.LoadFile(CsvFile);
            if (rows.Count == 0)
            {
                Debug.LogWarning($"[DungeonRegistry] {CsvFile} missing/empty — using built-in Vault fallback only.");
                _dungeons.Add(BuildVaultFallback());
                _byId[_dungeons[0].id] = _dungeons[0];
                return;
            }

            var seenIds = new HashSet<string>();
            for (int i = 0; i < rows.Count; i++)
            {
                var d = ParseRow(rows[i]);
                if (d == null) continue;
                if (string.IsNullOrEmpty(d.id))
                {
                    Debug.LogWarning($"[DungeonRegistry] row {i + 1}: missing id — skipping.");
                    continue;
                }
                if (!seenIds.Add(d.id))
                {
                    Debug.LogWarning($"[DungeonRegistry] duplicate id '{d.id}' at row {i + 1} — skipping.");
                    continue;
                }
                _dungeons.Add(d);
                _byId[d.id] = d;
            }

            _dungeons.Sort((a, b) => a.sortIndex.CompareTo(b.sortIndex));

            if (_dungeons.Count == 0)
            {
                Debug.LogWarning("[DungeonRegistry] no valid dungeons parsed — using built-in Vault fallback.");
                _dungeons.Add(BuildVaultFallback());
                _byId[_dungeons[0].id] = _dungeons[0];
            }
        }

        static DungeonDefinition ParseRow(Dictionary<string, string> row)
        {
            var d = new DungeonDefinition
            {
                id = CsvLoader.GetString(row, "id"),
                displayName = CsvLoader.GetString(row, "displayName"),
                sortIndex = CsvLoader.GetInt(row, "sortIndex"),
                themeColor = CsvLoader.ParseHexColor(CsvLoader.GetString(row, "themeColorHex"), Color.white),
                cardFocusRuleText = CsvLoader.GetString(row, "cardFocusRuleText"),
                cardPressureHintText = CsvLoader.GetString(row, "cardPressureHintText"),
                unlockGateDungeonId = CsvLoader.GetString(row, "unlockGateDungeonId"),
                unlockGateFloor = CsvLoader.GetInt(row, "unlockGateFloor"),
                enemyHpBase = CsvLoader.GetFloat(row, "enemyHpBase", 20f),
                enemyHpGrowth = CsvLoader.GetFloat(row, "enemyHpGrowth", 1.12f),
                enemyAtkBase = CsvLoader.GetFloat(row, "enemyAtkBase", 4f),
                enemyAtkGrowth = CsvLoader.GetFloat(row, "enemyAtkGrowth", 1.10f),
                enemyCountBase = CsvLoader.GetInt(row, "enemyCountBase", 2),
                enemyCountStep = CsvLoader.GetInt(row, "enemyCountStep", 4),
                goldBase = CsvLoader.GetFloat(row, "goldBase", 10f),
                goldGrowth = CsvLoader.GetFloat(row, "goldGrowth", 1.5f),
                floorClearTimeBase = CsvLoader.GetFloat(row, "floorClearTimeBase", 3f),
                floorClearTimePerFloor = CsvLoader.GetFloat(row, "floorClearTimePerFloor", 0.5f),
                anchorEnemyHpMultiplier = CsvLoader.GetFloat(row, "anchorEnemyHpMultiplier", 1f),
                misFocusDamageMultiplier = CsvLoader.GetFloat(row, "misFocusDamageMultiplier", 1f)
            };

            string focusRuleStr = CsvLoader.GetString(row, "focusRule");
            d.focusRule = ParseFocusRule(focusRuleStr);

            string unlockKindStr = CsvLoader.GetString(row, "unlockKind");
            d.unlockKind = ParseUnlockKind(unlockKindStr);

            return d;
        }

        static FocusRule ParseFocusRule(string s)
        {
            if (string.IsNullOrEmpty(s)) return FocusRule.Standard;
            switch (s.Trim())
            {
                case "Standard": return FocusRule.Standard;
                case "NoFocus": return FocusRule.NoFocus;
                case "ForcedFocus": return FocusRule.ForcedFocus;
                default:
                    Debug.LogWarning($"[DungeonRegistry] unknown focusRule '{s}' — defaulting to Standard.");
                    return FocusRule.Standard;
            }
        }

        static UnlockKind ParseUnlockKind(string s)
        {
            if (string.IsNullOrEmpty(s)) return UnlockKind.AlwaysUnlocked;
            switch (s.Trim())
            {
                case "AlwaysUnlocked": return UnlockKind.AlwaysUnlocked;
                case "ClearFloor": return UnlockKind.ClearFloor;
                default:
                    Debug.LogWarning($"[DungeonRegistry] unknown unlockKind '{s}' — defaulting to AlwaysUnlocked.");
                    return UnlockKind.AlwaysUnlocked;
            }
        }

        static DungeonDefinition BuildVaultFallback()
        {
            return new DungeonDefinition
            {
                id = "vault",
                displayName = "The Vault",
                sortIndex = 0,
                themeColor = new Color(0.61f, 0.55f, 0.44f),
                focusRule = FocusRule.Standard,
                cardFocusRuleText = "Focus an enemy to make your party concentrate fire. No focus → each hero chooses on their own.",
                cardPressureHintText = "A balanced dungeon. Bring whoever you've got.",
                unlockKind = UnlockKind.AlwaysUnlocked
            };
        }

        // Pure deterministic Tide-enemy designation. Same inputs → same index, across preview and live run.
        // FNV-1a 32-bit; do not replace with string.GetHashCode (runtime-randomized).
        public static int GetTideIndex(string dungeonId, int floor, int enemyCount)
        {
            if (enemyCount <= 0) return -1;
            uint hash = 2166136261u;
            if (!string.IsNullOrEmpty(dungeonId))
            {
                for (int i = 0; i < dungeonId.Length; i++)
                {
                    hash ^= dungeonId[i];
                    hash *= 16777619u;
                }
            }
            unchecked
            {
                hash ^= (uint)floor;
                hash *= 16777619u;
            }
            return (int)(hash % (uint)enemyCount);
        }
    }
}

+        IEnumerator ShakeCardRoutine(DungeonCard card)
+        {
+            if (card == null || card.root == null) yield break;
+            Vector2 home = card.root.anchoredPosition;
+            float dur = 0.35f;
+            float t = 0f;
+            while (t < dur)
+            {
+                t += Time.deltaTime;
+                float p = t / dur;
+                float ox = Mathf.Sin(p * Mathf.PI * 6f) * (1f - p) * 16f;
+                card.root.anchoredPosition = home + new Vector2(ox, 0);
+                yield return null;
+            }
+            card.root.anchoredPosition = home;
